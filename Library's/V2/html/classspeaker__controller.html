Detailed Description

Speaker controller class.

This class is used for controlling the speaker and converting the sound_lookup struct received from the sound pool to useable variables. Speaker_controller uses the concurrency mechanism pool with mutex for receiving which sound is needed to be played. It also uses a flag to know when a new sond is ready to be read from the pool and played.

Detailed Description

Lcd line structure.

This structure is used to make sure we can edit individual lines on the lcd. It contains 8 lines of 16 characters long. There are also some boolean values for making it easy to tell the controller you want one of three bigger texts on the lcd. These texts are "hit", "game begin" and "game end"
